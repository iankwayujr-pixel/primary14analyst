# Face ID Verification System

This repository contains a Face ID verification system implemented in Python. Below are the key components of the system.

## Components
- **face_id_system.py**: Main system class handling the overall functionality.
- **face_detector.py**: Code for detecting faces in images.
- **face_encoder.py**: Logic for encoding detected faces.
- **verification_engine.py**: Handles the verification of face IDs.
- **database.py**: Used for storing face data and managing access.
- **main.py**: Entry point for running the system.
- **config.py**: Configuration settings to customize system behavior.
- **requirements.txt**: Python dependencies for the project.

## Usage
1. Install dependencies from `requirements.txt`.
2. Run the system via `main.py`.
